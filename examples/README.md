# Demonstration projects

The three projects in this directory are **synthetic demonstration cases**:

- `nf-demo-service-7m.json` — approximately 7 m service launch;
- `nf-demo-patrol-10m.json` — approximately 10 m patrol launch;
- `nf-demo-rescue-12m.json` — approximately 12 m rescue/support craft.

They exercise the same integrated pipeline used by the API and user interface.
The generated PDF, DOCX, XLSX, CSV and JSON reports are examples of software
output and are not validated vessel designs, quotations or statutory documents.
